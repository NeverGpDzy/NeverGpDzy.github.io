完全二叉树叶子结点公式(待验证)

计算方法​

完全二叉树是一种特殊的二叉树，除了最后一层外，所有层的结点都是满的，并且最后一层的结点都尽可能地集中在左侧。

用一个简单的公式来计算叶子结点的数量：

L=⌈n+12⌉ L = \left\lceil \frac{n+1}{2} \right\rceil L=⌈2n+1​⌉

对于有 1001 个结点的完全二叉树：

L=⌈1001+12⌉=⌈10022⌉=⌈501⌉=501 L = \left\lceil \frac{1001 + 1}{2} \right\rceil = \left\lceil \frac{1002}{2} \right\rceil = \left\lceil 501 \right\rceil = 501 L=⌈21001+1​⌉=⌈21002​⌉=⌈501⌉=501

因此，这棵完全二叉树上有 501 个叶子结点。

结点数量与叶子结点数量的关系​

对于一棵完全二叉树：

  1. 层次结构：

    • 如果一棵完全二叉树有 hhh 层，那么前 h−1h-1h−1 层的结点数是满的，每层有 2层号2^{\text{层号}}2层号 个结点。
    • 最后一层的结点数可以从 1 到 2h−12^{h-1}2h−1 不等。

  2. 结点总数：

    • 前 h−1h-1h−1 层的结点总数是：1+2+4+⋯+2h−2=2h−1−11 + 2 + 4 + \cdots + 2^{h-2} = 2^{h-1} - 11+2+4+⋯+2h−2=2h−1−1。
    • 最后一层的结点数设为 kkk，则总结点数为：2h−1−1+k2^{h-1} - 1 + k2h−1−1+k。

  3. 叶子结点的数量：

    • 最后一层的结点都是叶子结点，其数量就是 kkk。
    • 由于完全二叉树的性质，前 h−1h-1h−1 层的结点数为 2h−1−12^{h-1} - 12h−1−1，这些都是非叶子结点。

公式推导​

设完全二叉树有 nnn 个结点，那么：

n=2h−1−1+k n = 2^{h-1} - 1 + k n=2h−1−1+k

其中 2h−12^{h-1}2h−1 是前 h−1h-1h−1 层的结点数，kkk 是最后一层的结点数。

根据完全二叉树的性质，最后一层的结点数 kkk 满足：

1≤k≤2h−1 1 \leq k \leq 2^{h-1} 1≤k≤2h−1

我们需要找到 kkk 的表达式。实际上，最后一层的结点数 kkk 就是叶子结点的数量 LLL：

L=k=n−(2h−1−1) L = k = n - (2^{h-1} - 1) L=k=n−(2h−1−1)

但是，我们更常用一个简单的公式来计算叶子结点的数量：

L=⌈n+12⌉ L = \left\lceil \frac{n+1}{2} \right\rceil L=⌈2n+1​⌉

公式解释​

这个公式的推导可以通过观察完全二叉树的性质来理解：

  1. 完全二叉树的结点分布：

    • 如果完全二叉树有 nnn 个结点，那么它的叶子结点数 LLL 和非叶子结点数 III 满足：
    n=L+I n = L + I n=L+I
    • 由于每个非叶子结点至少有一个子结点，所以：
    I=n−L I = n - L I=n−L

  2. 非叶子结点的数量：

    • 在完全二叉树中，非叶子结点数 III 总是比叶子结点数 LLL 小 1 或相等。因此：
    I=L−1 I = L - 1 I=L−1

  3. 结合总结点数：

    • 通过结合以上关系，我们有：
    n=L+(L−1) n = L + (L - 1) n=L+(L−1) n=2L−1 n = 2L - 1 n=2L−1
    • 解这个方程，我们得到：
    L=n+12 L = \frac{n + 1}{2} L=2n+1​

由于 LLL 必须是整数，我们取上取整：

L=⌈n+12⌉ L = \left\lceil \frac{n+1}{2} \right\rceil L=⌈2n+1​⌉

应用​

对于有 1001 个结点的完全二叉树：

L=⌈1001+12⌉=⌈10022⌉=⌈501⌉=501 L = \left\lceil \frac{1001 + 1}{2} \right\rceil = \left\lceil \frac{1002}{2} \right\rceil = \left\lceil 501 \right\rceil = 501 L=⌈21001+1​⌉=⌈21002​⌉=⌈501⌉=501

因此，这棵完全二叉树上有 501 个叶子结点。
