威海 Weihai

回头看这组照片，威海留给我的并不是一串被依次完成的地点，而是一条冬天海岸线不断换表情的过程。前一段还是荣成偏冷、偏空的海面，下一段已经变成布鲁维斯号旁边那种迎面压过来的风；再往后，冰柱、海鸥、日出、渔船和乳山大集里快过年的热气，又把这趟旅行一点点接了下去。

从华星宾馆那片海，开过烟墩角，再到天鹅湖​

第一组照片写的是从华星宾馆一带到烟墩角、再到天鹅湖这一段。华星宾馆一带的树影、潮水和岸线都很淡，往烟墩角开的路上，鱼干、村子和冷风把海边的日常带了出来；等到天鹅湖出现，冰面、芦苇和白天鹅又让这一段慢了下来。

布鲁维斯号把风都留在了海边​

布鲁维斯号那一带几乎把海边最硬的一面都摆出来了。雪还留在沙滩上，浪一层层往前推，海鸥被风托着，一会儿压低到离人很近，一会儿又抬高，从船和海之间来回掠过去。那艘船停在远处，风却始终不肯停，所以这些照片看起来不只是冷，更有一种海正面压过来的力度。

孤独公约附近，海更汹涌了​

到了孤独公约附近，海没有安静下来，反而看起来更汹涌了。站在高处往下看，浪不断往礁石和岸边卷，灰白色的海面被风推得一层接一层，整片海都像还在往前逼近。前面在布鲁维斯号那边感受到的是迎面而来的硬风，这里看到的则是海本身被彻底掀起来的样子。

路上那袋野生渤海湾大明虾​

就这一张照片，已经足够把那次停下来的记忆钉住了：十块钱一只，贵得让人当场就记住。海边旅行里常常会有这种时候，前面刚看完海，转头手里已经拎着当天买下的海味。

葡萄滩把冬天冻成了冰柱​

葡萄滩海水浴场的冰柱，是威海冬天最直白的一面。海还是在动的，岸上却已经先冻住了，栏杆、步道和水边的边缘都像被重新塑了一遍。很多地方的冬天只是让人知道气温低，但威海在这里会直接把“冷”做成看得见的形状。

火炬八街一路落到国际海水浴场​

火炬八街真正吸引人的地方，从来不只是那条街本身，而是它和海的关系太直接。顺着街道往下走，视线会自然落到威海国际海水浴场；等真正走到沙滩上，海风、人流和浪会把前面那种一路向海的感觉变得更完整。冬天到了这里，海边也还是热闹的。

九龙湾的海冰、海鸥和傍晚​

九龙湾沙滩浴场这一段给人的感觉最完整。先是近岸的海冰，把海边压出一层很薄的亮色；再往后，海鸥开始低低地从人头上掠过去，翅膀几乎擦着风和浪，成群地在海冰上方盘旋，整片海边立刻有了动静；至海港湾那片很特别的海上足球场也让城市边缘出现在画面里。等到傍晚的粉蓝色落下来，这一段才慢慢安静下来。

日出以后，再去海源公园看渔船​

看日出和去海源公园，是威海早晨很好的一种打开方式。太阳慢慢把海面和渔船照亮，礁石也跟着变得更清楚。清晨的海边没有白天那么多声音，但画面反而更满，整座城市像是先把节奏放慢了一点。

乳山让这趟旅行多停了一会儿​

乳山这一段像是让尾声多停了一会儿。沿海公路一直贴着海往前走，到了生蚝最有名的地方，海的味道从风里换到了摊位和餐桌上；第二天再去赶大集，年货、热闹和快过年的气息又让这趟旅行从海边回到生活里。

回头再看，真正留下来的也不是“我去了几个点”，而是冬天海岸线不断换表情的过程：天鹅湖的安静、布鲁维斯号那边贴着风盘旋的海鸥、孤独公约附近更汹涌的海、葡萄滩的冰柱、九龙湾那片被海鸥重新搅热的海边、海源公园清晨的渔船，还有乳山大集里那种快过年的热气。它们放在一起，才是这趟威海。
