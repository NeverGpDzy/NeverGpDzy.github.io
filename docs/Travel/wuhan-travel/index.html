武汉 Wuhan

武汉给人的第一感觉不是某一个明确的地标，而是一种很容易把人留下来的尺度。街道并不急着把一切塞满，水边、桥下、转角和晚风都像在提醒你，这座城市适合边走边看，而不是匆忙打卡。

这组照片更像一次慢慢靠近的过程。镜头没有特别追求完整叙事，只是把那些在路上忽然停下来的瞬间留了下来，让一座城市从远处的印象，慢慢变成可以反复回想的细节。

把脚步放慢一点​

真正开始喜欢一座城市，往往不是因为它有多强烈，而是因为它愿意给人留出一点空白。路边的树影、天色的变化、行人和建筑之间的距离，看久了以后会让人不自觉地放慢脚步。

拍照的时候也能感觉到这种节奏。很多画面并不需要刻意寻找，往前走几步，或者在原地多停一会儿，城市就会自己把层次摆出来。武汉给我的就是这种不需要抢拍、却总有东西可看的松弛感。

夜色没有结束这趟路​

到了晚上，这座城没有突然切换成完全不同的样子，它只是把白天的轮廓收得更深了一点。灯光落下来之后，街道、江边和远处的建筑开始有了另一种安静，像是同一段路被重新读了一遍。

相册最后留下这些夜色，也不是为了制造收尾的仪式感，而是因为很多旅行的记忆，恰恰是在天黑以后才真正沉下来。白天是看见，夜里才开始记住。

离开之后再回看这些照片，我记住的不是某一个“必须去”的地方，而是武汉那种既有分量、又不逼人的城市气息。它不会急着证明自己，却很容易在很多细小的瞬间里留下来。
