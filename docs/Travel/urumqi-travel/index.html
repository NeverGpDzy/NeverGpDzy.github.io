乌鲁木齐 Urumqi

乌鲁木齐这趟相册里，最先出现的不是某一处景观，而是“路”本身。越往前走，越能感觉到空间被一点点拉开，城市和风景的边界不像在很多地方那样清晰，它们更像是顺着地势和视线自然过渡过去。

这种开阔会改变人看东西的方式。你不太会急着盯住一个点，反而会先看见远处的轮廓、天色的变化、车窗外不断后退的线条，然后才慢慢注意到那些更贴近眼前的细节。这组照片，也是沿着这种越来越远的感觉展开的。

路先把人带远​

旅行里有些地方，是到了之后才开始看；但也有些地方，从出发那一刻就已经进入状态。乌鲁木齐给我的感觉更接近后者。路上的风、光线、地形和距离感，本身就是这趟旅行非常重要的一部分。

很多画面之所以会被拍下来，不是因为它们特别戏剧化，而是因为那种路途感太明确了。人会在不断变化的远近之间，慢慢意识到自己已经离熟悉的尺度越来越远，眼前的一切也因此变得更值得记录。

开阔比想象更具体​

真正站在这样的地方，才会知道“开阔”不是一句很抽象的形容。它是风吹过来时的方向，是天空压下来的高度，是远处一直延伸却看不完的层次。那些在照片里看似平静的画面，现场往往都带着更直接的力量。

我很喜欢这种力量不是通过喧闹表达出来的方式。没有太多需要强调的地方，也没有必须制造出的高潮，景色本身就已经足够完整。你只要站着，看它慢慢展开，然后在合适的时候按下快门。

回头再看这些照片，记住的首先还是那种被空间彻底打开的感觉。乌鲁木齐这一段旅行留下来的，不只是风景本身，还有一路向远处走去时，人也被一起拉开、放松、重新校准的过程。
