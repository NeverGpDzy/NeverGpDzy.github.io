docker常用命令

查看镜像列表​

docker images

启动容器​

docker run

查看容器​

docker ps <container_name_or_id>

包括停止状态的容器

docker ps -a <container_name_or_id>

停止容器​

docker stop <container_name_or_id>
docker kill <container_name_or_id>

重启容器​

docker start <container_name_or_id>

删除容器​

docker rm -f <container_name_or_id>

删除镜像​

docker rmi <image_name_or_id>

查看日志​

docker logs 相应的容器名字
