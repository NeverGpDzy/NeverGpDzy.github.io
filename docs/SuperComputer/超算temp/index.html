conda_txt迁移环境​

  1. 迁移环境

一旦你准备好将环境迁移到有 NVIDIA GPU 的机器上，你有几个选项：

  • 复制虚拟环境：你可以简单地将整个虚拟环境文件夹复制到目标机器上。然而，这种方法可能会因为不同操作系统或架构而遇到兼容性问题。

  • 导出依赖列表：更可靠的方法是导出当前环境的依赖列表，然后在目标机器上重新创建虚拟环境并安装依赖。

    • 使用 pip:

      pip freeze > requirements.txt

    • 使用 conda:

      conda list --export > requirements.txt

  • 在目标机器上，根据导出的 requirements.txt 文件重新创建虚拟环境并安装依赖。请确保在目标机器上安装了与模型兼容的 NVIDIA 驱动、CUDA Toolkit 和 cuDNN。
